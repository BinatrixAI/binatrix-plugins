# Monday.com Lite Plugin

A lightweight Monday.com management plugin for [Claude Code](https://claude.com/product/claude-code). Same operational coverage as the full `monday-com` plugin, optimized for ~70% fewer tokens.

## Installation

```bash
/plugin install monday-com-lite@binatrix-plugins
```

## What It Does

Same capabilities as the full monday-com plugin, compressed for minimal context window usage:

- **Board & Item CRUD** — All core operations
- **CRM Architecture** — 4-board setup patterns
- **Column Value Formatting** — All 30+ column types
- **Dashboards, Automations, Workspaces** — Templates and patterns
- **Error Handling & Rate Limits** — Complete reference

## When to Use This vs Full Version

| | monday-com | monday-com-lite |
|---|---|---|
| Token usage | ~10,500 | ~3,070 |
| Coverage | Complete | Complete |
| Examples | Detailed | Compressed |
| Edge cases | Inline | In references |

Use **lite** for everyday operations. Use **full** when you need extensive examples or edge case documentation inline.
